Q1: What is cloud computing and why is it important?

A: Cloud computing delivers computing services over the internet, eliminating the need for companies to maintain data centers. It reduces costs through pay-as-you-use pricing and provides scalable resources on-demand.

Q2: What are the main differences between public, private, and hybrid cloud?

A: Public cloud is shared infrastructure by third parties (AWS, GCP), private cloud is dedicated infrastructure for one organization, and hybrid cloud combines both - using private for sensitive data and public for non-critical workloads.
