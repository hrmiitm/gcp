🎯 Interview Questions & Answers

❓ Basic Cloud Computing Questions

Q1: What is cloud computing and why is it important?

A: Cloud computing delivers computing services over the internet, eliminating the need for companies to maintain data centers. It reduces costs through pay-as-you-use pricing and provides scalable resources on-demand.

❓ Cloud Models Comparison

Q2: What are the main differences between public, private, and hybrid cloud?

A: Public cloud is shared infrastructure by third parties (AWS, GCP), private cloud is dedicated infrastructure for one organization, and hybrid cloud combines both - using private for sensitive data and public for non-critical workloads.

❓ GCP Specific Questions

Q3: Why choose GCP over AWS or Azure?

A: GCP offers proven scalability (powers Gmail/YouTube), strong AI/ML capabilities, competitive pricing, and Google’s global network. AWS has largest market share, while Azure integrates well with Microsoft ecosystems.

❓ Technical Concepts

Q4: What is virtualization in cloud computing?

A: Virtualization allows multiple virtual machines to run on a single physical server using a hypervisor. It enables cloud providers to efficiently share physical resources among multiple customers while maintaining isolation.

❓ Free Tier Questions

Q5: Explain GCP’s free tier offerings.

A: GCP offers $300 in free credits for 90 days for new users, plus always-free products including 1 E2-micro instance, 5GB cloud storage, and limited usage of 20+ services. Charges apply only when exceeding these limits.
